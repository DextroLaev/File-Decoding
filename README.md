# File-Decoding
Decoding of BMP image file and WAVE music file

For BMP image:

    For python:
    
        $  python3 bmp_new.py
        
    For C:
    
        $  gcc decode.c decodeApi.c -o out
        $  ./out
    
  
For WAVE music:

    $  python3 audioExtract.py
  
    $  aplay -f cd -r 44100 raw
